# Habib Achour Advisory

A premium static website for Habib Achour, strategic advisor in copyright, collective management organizations and creative-industry development.

## Stack

- Static HTML/CSS/JS
- Vercel deployment
- Warm-orange design system

## Local development

```bash
npm run dev
```

## Build

```bash
npm run build
```

The build output is copied to `dist/` for Vercel.
